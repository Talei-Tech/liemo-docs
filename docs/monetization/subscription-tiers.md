# Monetization Model

## Subscription Tiers

| Feature | Free | Pro (USD 9.99/mo) | Enterprise (USD 29.99/mo) |
|---------|------|------------------|---------------------------|
| Store Profile & Links | ✓ | ✓ | ✓ |
| Number of Platform Links | 3 | Unlimited | Unlimited |
| Categories | 1 | 3 | Unlimited |
| Basic Analytics (views, clicks) | Last 7 days | Last 90 days | Full history |
| Platform Breakdown Analytics | ✗ | ✓ | ✓ |
| Verified Badge | ✗ | ✓ | ✓ |
| Boost / Featured Listing | ✗ | Wallet credits | Priority + credits |
| Ads Shown to Followers | Yes (platform ads) | No ads | No ads |
| CSV Export | ✗ | ✗ | ✓ |
| Priority Support | ✗ | Email | Email + Chat |

## Wallet & Credits

- Store admins top up wallet via PayMongo / Stripe / KNET
- Credits used for: Boost Store (appear in 'Featured' for 24h), Send Push Notification to followers, Promoted category placement
- 1 credit = USD 1 (or regional equivalent); minimum top-up 10 credits

## Advertising (Free End User Tier)

- Banner ads shown between store listings in browse/search feeds on mobile
- Native sponsored store cards (labeled 'Sponsored') in category results
- Managed by super admin via the Ads dashboard
- CPM and CPC pricing models; third-party brands can self-serve (Phase 2)
