# GitHub Projects Setup Guide

## Recommended Board Structure

Use GitHub Projects (v2) with the following views:

- **Backlog View (Table)** — All issues sorted by priority; columns: Title, Epic, Priority, Points, Sprint, Assignee
- **Sprint Board (Board/Kanban)** — Columns: Backlog, Todo, In Progress, In Review, Done
- **Roadmap View** — Timeline view by Epic/Milestone across quarters
- **Bug Tracker (Table)** — Filtered by 'bug' label; columns: Title, Severity, Reporter, Assignee, Status
